interaction.exception.master¶

MasterErrorCode¶

enum nervex.interaction.exception.master.MasterErrorCode(value)[source]¶
Overview:

Error codes for master end

Member Type

int

Valid values are as follows:

SUCCESS = <MasterErrorCode.SUCCESS: 0>¶

Master request success

SYSTEM_SHUTTING_DOWN = <MasterErrorCode.SYSTEM_SHUTTING_DOWN: 101>¶

Master end is shutting down

CHANNEL_NOT_GIVEN = <MasterErrorCode.CHANNEL_NOT_GIVEN: 201>¶

No channel id given in request

CHANNEL_INVALID = <MasterErrorCode.CHANNEL_INVALID: 202>¶

Channel id given not match with master end

MASTER_TOKEN_NOT_GIVEN = <MasterErrorCode.MASTER_TOKEN_NOT_GIVEN: 301>¶

No master token found in connection request from slave

MASTER_TOKEN_INVALID = <MasterErrorCode.MASTER_TOKEN_INVALID: 302>¶

Master token auth failed in master end

SELF_TOKEN_NOT_GIVEN = <MasterErrorCode.SELF_TOKEN_NOT_GIVEN: 401>¶

No self token given in self request (such as ping, shutdown)

SELF_TOKEN_INVALID = <MasterErrorCode.SELF_TOKEN_INVALID: 402>¶

Self token auth failed in master end itself (such as ping, shutdown)

SLAVE_TOKEN_NOT_GIVEN = <MasterErrorCode.SLAVE_TOKEN_NOT_GIVEN: 501>¶

No slave token given in service request from slave

SLAVE_TOKEN_INVALID = <MasterErrorCode.SLAVE_TOKEN_INVALID: 502>¶

Slave token not found in master end

TASK_DATA_INVALID = <MasterErrorCode.TASK_DATA_INVALID: 601>¶

Task data is invalid

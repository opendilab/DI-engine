interaction.exception.slave¶

SlaveErrorCode¶

enum nervex.interaction.exception.slave.SlaveErrorCode(value)[source]¶
Overview:

Error code for slave end

Member Type

int

Valid values are as follows:

SUCCESS = <SlaveErrorCode.SUCCESS: 0>¶

Slave request success

SYSTEM_SHUTTING_DOWN = <SlaveErrorCode.SYSTEM_SHUTTING_DOWN: 101>¶

Slave end is shutting down

CHANNEL_NOT_FOUND = <SlaveErrorCode.CHANNEL_NOT_FOUND: 201>¶

No channel id given in request

CHANNEL_INVALID = <SlaveErrorCode.CHANNEL_INVALID: 202>¶

Channel id given not match with slave end

MASTER_TOKEN_NOT_FOUND = <SlaveErrorCode.MASTER_TOKEN_NOT_FOUND: 301>¶

No master token found in connection request from master

MASTER_TOKEN_INVALID = <SlaveErrorCode.MASTER_TOKEN_INVALID: 302>¶

Master token auth failed in slave end

SELF_TOKEN_NOT_FOUND = <SlaveErrorCode.SELF_TOKEN_NOT_FOUND: 401>¶

No self token given in self request (such as ping, shutdown)

SELF_TOKEN_INVALID = <SlaveErrorCode.SELF_TOKEN_INVALID: 402>¶

Self token auth failed in slave end itself (such as ping, shutdown)

SLAVE_ALREADY_CONNECTED = <SlaveErrorCode.SLAVE_ALREADY_CONNECTED: 501>¶

Slave end has already connected to another master end

SLAVE_NOT_CONNECTED = <SlaveErrorCode.SLAVE_NOT_CONNECTED: 502>¶

Slave end not connected with master end yey

SLAVE_CONNECTION_REFUSED = <SlaveErrorCode.SLAVE_CONNECTION_REFUSED: 503>¶

Connection to slave end refused

SLAVE_DISCONNECTION_REFUSED = <SlaveErrorCode.SLAVE_DISCONNECTION_REFUSED: 504>¶

Disconnection to slave end refused

TASK_ALREADY_EXIST = <SlaveErrorCode.TASK_ALREADY_EXIST: 601>¶

Slave end is processing another task

TASK_REFUSED = <SlaveErrorCode.TASK_REFUSED: 602>¶

Task for slave end refused

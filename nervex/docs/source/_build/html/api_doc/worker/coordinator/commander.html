finish_collector_task(task_id: str, finished_task: dict) → bool[source]¶
Overview:

Get collector’s finish_task_info and release collector_task_space. If collector’s task is evaluation, judge the convergence and return it.

Arguments:

  • task_id (str): the collector task_id

  • finished_task (dict): the finished task

Returns:
  • convergence (bool): Whether the stop val is reached and the algorithm is converged.

    If True, the pipeline can be finished.

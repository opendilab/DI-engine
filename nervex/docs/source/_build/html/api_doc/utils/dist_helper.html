utils.dist_helper¶

dist_helper¶

Please Reference nerveX/nervex/utils/disk_helper.py for usage.

errro_wrapper¶

Overview:

wrap the function, so that any Exception in the function will be catched and return the default_ret

Arguments:

  • fn (Callable): the function to be wraped

  • default_ret (obj): the default return when an Exception occured in the function

Returns:

  • wrapper (Callable): the wrapped function

Examples:
>>> # Used to checkfor Fakelink (Refer to utils.dist_helper.py)
>>> def get_rank():  # Get the rank of linklink model, return 0 if use FakeLink.
>>>    if is_fake_link:
>>>        return 0
>>>    return error_wrapper(link.get_rank, 0)()

get_rank¶

Overview:

Get the rank of linklink model, return 0 if use FakeLink.

Note

Reference import_helper.try_import_link and linklink.get_rank.

get_world_size¶

Overview:

Get the world_size of linklink model, return 0 if use FakeLink.

Note

Reference import_helper.try_import_link and linklink.get_world_size.

broadcast¶

Overview:

Use linklink.broadcast and raise error when using FakeLink

Arguments:

  • value (obj): the value to board cast

  • rank_num (int): the rank to boardcast on

allreduce¶

Overview:

Call linklink.allreduce on the data

Arguments:

  • data (obj): the data to reduce

  • op (str): the operation to perform on data, support ['sum', 'max']

get_group¶

Overview:

Get the group segmentation of group_size each group

Arguments:

  • group_size (int) the group_size

distributed_mode¶

Overview:

Wrap the function so that in can init and finalize automatically before each call

dist_init¶

Overview:

Init the distribution

Arguments:

  • method (str): Support ['slurm', 'single_node`]

  • device_id (int): Default device when using single_node method

dist_finalize¶

Overview:

Finalize linklink, see linklink.finalize()

simple_group_split¶

Overview:

Split the group according to worldsize, rank and num_groups

Note

With faulty input, raise array split does not result in an equal division
